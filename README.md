## ❓ Troubleshooting common issues

If the software does not open, check your background apps. Some programs block the installation. Restart your computer if you get an error during the setup. Make sure your graphics card drivers have the latest updates. You can find these updates on the website of your graphics card manufacturer. If the video playback stutters, lower the preview resolution in the playback settings menu. This makes the interface move faster.

## 📁 Managing project files

Store your video files in one folder. This keeps the project organized. If you move your clips to a different location on your computer, the software might show a "Media Offline" warning. To fix this, right-click the missing file and select "Relink Media." Point the software to the new location of the files. The link restores itself.

## 💡 Best practices for smooth performance

High-resolution video uses significant hardware resources. Close other programs while you edit. This leaves more memory for the video engine. Save your work often by pressing "Ctrl + S." This prevents data loss if your power goes out or your computer restarts. Keep your drive clean. If your hard drive becomes full, the software slows down. Remove old unused files to regain speed. Use an external drive for large video projects to save space on your main system drive.
